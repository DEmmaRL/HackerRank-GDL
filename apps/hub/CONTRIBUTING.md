# Contributing

Thank you for your interest in contributing to HackerRank Crew GDL.

## Ways to contribute

- Fix typos or improve documentation.
- Propose content updates, ammenments.
- Propose article entries.
- Improve accessibility, performance, and UI consistency.
- Report bugs and suggest enhancements.

## Development setup

1. Fork this repository.
2. Create a branch with a clear name.
3. Install dependencies:

```bash
pnpm install
```

4. Start local development:

```bash
pnpm dev
```

5. Validate your changes:

```bash
pnpm build
```

## Pull request checklist

- Keep changes focused and small.
- Use clear commit messages.
- Confirm the site builds successfully.
- Update docs/content when behavior changes.
- Follow existing code style and structure.

## Content guidelines

- Use respectful and inclusive language.
- Keep technical claims accurate and sourced when possible.
- Avoid sharing private or sensitive personal data.

## Questions

If you are unsure about a contribution direction, open an issue first to discuss scope and approach.
