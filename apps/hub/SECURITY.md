# Security Policy

## Supported versions

This project is actively maintained on the `main` branch.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these options instead:

1. Open a private GitHub Security Advisory in this repository.
2. If private advisories are unavailable, open an issue with minimal details and request a private contact channel.

We will acknowledge reports as soon as possible and work on a fix based on severity and impact.
